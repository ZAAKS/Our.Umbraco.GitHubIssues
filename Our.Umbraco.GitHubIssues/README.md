## Features

* **Displays open Umbraco GitHub issues** in your backoffice.
* **Click on an issue** to open it directly on GitHub.
* **Keeps you up-to-date** without leaving Umbraco.
* **All accessible** under **Settings ‚Üí GitHub Issues**.
* **No complicated setup**, just install and go.

---

## Installation

You can install via **NuGet**:

```powershell
Install-Package Our.Umbraco.GitHubIssues
```

Or grab it from the [NuGet Gallery](https://www.nuget.org/packages/Our.Umbraco.GitHubIssues/).

After installation, a **GitHub Issues** dashboard will appear in the **Settings** section of your Umbraco backoffice.
